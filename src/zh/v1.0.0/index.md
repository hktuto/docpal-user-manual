# 介绍
DocPal 是一款在线文档管理工具，较传统文档管理系统加入文档审核功能、开放式上传及分享。支持文档多维度搜索，支持文档操作历史查看。 用户仅需维护一个文件树即可保存及快速查看所有文件。
